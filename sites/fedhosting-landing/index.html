Why it is different

Built so no one can take you down

🔗

Federated network

Your website lives on multiple independent servers around the world. If one goes down, the others keep serving your site without interruption.

🔑

You own your content

There is no central company with the power to remove your website. The network is run by many independent operators, not one corporation.

⚡

Fast everywhere

Because your site is replicated across the network, visitors get served from the closest node. Pages load quickly no matter where your audience is.

🆓

Free to start

Upload up to 500 MB and deploy as many times as you like — no credit card required. Larger plans are available when you need more.

🛡️

Secure by default

Every server in the network uses HTTPS. Every node verifies the identity of every other node using cryptographic signatures — like passkeys for servers.

📦

Simple file format

No complicated setup. Just upload HTML, CSS, JavaScript, images, and fonts. If it runs in a browser, it works here.
